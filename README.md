# YouTube Clone Frontend

This repository contains a frontend practice project that recreates a YouTube-style homepage layout using HTML and CSS.

## Live Demo

- Main project entry: https://omshejwal.github.io/Youtube/
- YouTube clone page: https://omshejwal.github.io/Youtube/lec7/youtube.html

## Project Structure

- `lec7/youtube.html` - Main YouTube clone page
- `lec7/styles/` - Split CSS files for layout and components
- `lec7/thumbnail/` - Icons, channel images, and thumbnails
- `.github/workflows/pages.yml` - GitHub Pages deployment workflow

## Run Locally

1. Clone the repository.
2. Open the project folder in VS Code.
3. Open `lec7/youtube.html` in your browser.

## Deployment

The project is automatically deployed with GitHub Pages using GitHub Actions on every push to the `main` branch.

## Notes

- Image paths in `lec7/youtube.html` are configured as relative paths for web deployment.
- If updates do not appear immediately on the live site, hard refresh the browser (`Ctrl+Shift+R`).
